# 12. Alien Dictionary (LC 269 / GFG) — Hard
🔗 [Open on LeetCode](https://leetcode.com/problems/alien-dictionary/) · [GFG](https://www.geeksforgeeks.org/problems/alien-dictionary/1)

**Pattern:** Graph — **topological sort** (derive an ordering from *pairwise* constraints)
**Trigger:** "words are **sorted** in an **unknown alphabet**; recover the **letter order**" → each **adjacent word pair** gives **one** edge (first differing char: earlier→later); topo-sort the letters. Cycle ⇒ contradiction ⇒ `""`.

## Problem (1-liner)
Given `words[]` sorted lexicographically in an alien alphabet, return **any** valid order of its distinct letters, or **`""`** if the ordering is impossible (contradiction).
**Examples:** `["baa","abcd","abca","cab","cad"]` → `"bdac"` · `["caa","aaa","aab"]` → `"cab"` · `["ab","cd","ef","ad"]` → `""` (a<e from pair 1, e<a from pair 3 → cycle).
**Constraints:** `1 ≤ words.length ≤ 500`, `1 ≤ words[i].length ≤ 100`, lowercase English letters.

## Idea (1-liner)
Only **consecutive** words constrain the order, and each pair gives exactly **one** relation — the **first position where they differ**: `s1[i]` comes **before** `s2[i]`. Build a DAG over the 26 letters from those edges and topo-sort (Kahn's BFS). If not all present letters can be peeled, there's a **cycle** → `""`.

> **Load-bearing insight:** don't compare whole words or every pair. Sortedness means the *only* information in `("abcd","abca")` is the **single** edge `d → a` at the first mismatch — stop at the first differing char. **Two silent traps:** (1) the **prefix rule** — if `s1` is longer than `s2` yet `s2` is a prefix of `s1` (`["abc","ab"]`), that's **invalid** → `""`; (2) index the graph by **letter (`ch-'a'`, 0–25)**, never by "count of distinct letters", or you index out of bounds.

## Approach (Kahn's BFS)
1. **Collect present letters** — mark which of the 26 actually appear (only these belong in the answer).
2. **Build edges from adjacent pairs:** for `words[i], words[i+1]`, scan to the **first differing char** → edge `s1[j] → s2[j]`, `indegree[s2[j]]++`, then **break**.
3. **Prefix guard:** if you reach the end of the shorter word with **no** difference **and** `s1` is longer → return `""`.
4. **Topo-sort:** seed queue with **present** indegree-0 letters; peel, append to result, decrement dependents.
5. **Answer:** result length == number of present letters ⇒ return it; else a cycle blocked some letters ⇒ `""`.

## Code (clean canonical — Kahn's BFS)
```cpp
class Solution {
public:
    string findOrder(vector<string>& words) {
        vector<int> present(26, 0);
        for (auto& s : words)
            for (char ch : s) present[ch - 'a'] = 1;

        vector<vector<int>> adj(26);
        vector<int> indeg(26, 0);

        for (int i = 0; i + 1 < (int)words.size(); ++i) {
            string &a = words[i], &b = words[i + 1];
            int len = min(a.size(), b.size()), j = 0;
            for (; j < len; ++j) {
                if (a[j] != b[j]) {                 // first mismatch → one edge
                    adj[a[j] - 'a'].push_back(b[j] - 'a');
                    indeg[b[j] - 'a']++;
                    break;
                }
            }
            if (j == len && a.size() > b.size()) return "";   // prefix rule → invalid
        }

        queue<int> q;
        int total = 0;
        for (int i = 0; i < 26; ++i)
            if (present[i]) { ++total; if (indeg[i] == 0) q.push(i); }

        string res;
        while (!q.empty()) {
            int node = q.front(); q.pop();
            res += char('a' + node);
            for (int nbr : adj[node])
                if (--indeg[nbr] == 0) q.push(nbr);
        }
        return (int)res.size() == total ? res : "";   // short ⇒ cycle
    }
};
```
**TC:** O(N·L + V + E) — N words × L length to build, V ≤ 26, E ≤ N.  **SC:** O(V + E) — adjacency + indegree + queue.

1. **Missed the prefix rule** — `["abc","ab"]` produced `"abc"` instead of `""`; no branch handled "ran out of chars with no mismatch and `s1` is longer".
2. **Sized the graph by *distinct count*, indexed by *letter*** — `int size = st.size(); vector adjList(size); ... adjList[ch-'a']`. Works only when letters are contiguous from `'a'`; for `["yz","yx"]` (`'y'-'a'=24` on a size-2 vector) it's **out of bounds → UB/crash**. Everything must be sized to **26** with a `present[]` mask, and `res.size()` compared to the **count of present letters**.

Both are folded into the canonical block above.

## Other Approaches
- **Kahn's BFS (indegree peel)** *(canonical)* — iterative, cycle detection is free (short output ⇒ cycle). Default interview pick. O(N·L+V+E) / O(V+E).
- **DFS post-order + reverse** — 3-color to catch a back edge (⇒ `""`), collect post-order, reverse it. Same complexity, O(V) stack; reach for it if asked recursively. Mirror of [[7. 210. Course Schedule II]]'s DFS variant.
- **Brute force (no contender)** — repeatedly scan for a letter with no incoming edge, emit, remove edges, repeat: Kahn's in O(V²) instead of O(V+E). Mention only to contrast.

**Family map:** direct application of the topo-sort engine from [[6. 207. Course Schedule]] / [[7. 210. Course Schedule II]] — same Kahn's peel + cycle-as-short-output. The **new idea** is *deriving the edges* from sorted words (one edge per adjacent pair) rather than being handed them. Algorithm reference: [[00. Graph_Algo]].

## Pitfalls
- **One edge per adjacent pair, at the first mismatch** — don't add edges for later chars or non-adjacent pairs; `break` after the first difference.
- **Prefix rule** — `s1` longer than `s2` with `s2` a prefix ⇒ **invalid** (`""`). Easiest case to forget; no edge signals it, so guard explicitly.
- **Index by letter (0–25), size to 26** — sizing structures by "number of distinct letters" but indexing by `ch-'a'` is out-of-bounds UB. Use a `present[]` mask; only present letters enter the answer.
- **Cycle ⇒ `""`** — result length must equal the count of *present* letters; contradictions (`a<e` and `e<a`) leave letters stuck above indegree 0.
- **Only distinct letters in the output** — mask by presence; letters never appearing must not be emitted.
- **`words.size()-1` underflow** — loop as `i+1 < words.size()` (or cast), since `size()` is unsigned.
