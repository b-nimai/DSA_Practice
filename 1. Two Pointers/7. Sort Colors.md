# 7. Sort Colors (LC 75) — Medium
🔗 [Open on LeetCode](https://leetcode.com/problems/sort-colors/)

**Pattern:** Two Pointers — 3-way partition (Dutch National Flag)
**Trigger:** in-place sort with 3 distinct values / partition into <, =, >

## Problem (1-liner)
Sort array of `0/1/2` in-place in one pass without library sort.
**Example:** `[2,0,2,1,1,0]` → `[0,0,1,1,2,2]`

## Idea (1-liner)
Three regions: `[0..lo)` zeros, `[lo..mid)` ones, `(hi..n)` twos. Walk `mid`; swap to fix `0` or `2`.

## Approach
1. `lo = 0`, `mid = 0`, `hi = n - 1`.
2. While `mid <= hi`:
   - `nums[mid] == 0` → `swap(nums[lo++], nums[mid++])`
   - `nums[mid] == 1` → `mid++`
   - `nums[mid] == 2` → `swap(nums[mid], nums[hi--])` *(don't ++mid: unknown swapped-in value)*

## Code (clean canonical)
```cpp
void sortColors(vector<int>& nums) {
    int lo = 0, mid = 0, hi = nums.size() - 1;
    while (mid <= hi) {
        if (nums[mid] == 0)      swap(nums[lo++], nums[mid++]);
        else if (nums[mid] == 1) mid++;
        else                     swap(nums[mid], nums[hi--]);
    }
}
```
**TC:** O(n)  **SC:** O(1)  *(single pass)*

## My Solution
```cpp
class Solution {
public:
    void sortColors(vector<int>& nums) {
        int n = nums.size();
        int zero = 0, two = n - 1, ind = 0;
        while(ind <= two) {
            if(nums[ind] == 0) {
                swap(nums[ind], nums[zero]);
                ind++, zero++;
            } else if(nums[ind] == 2) {
                swap(nums[ind], nums[two]);
                two--;
            } else {
                ind++;
            }
        }
    }
};
```

## Other Approaches
- **Counting sort:** count 0/1/2, rewrite array → O(n) / O(1) (two passes)
- **Library sort:** `sort(nums.begin(), nums.end())` → O(n log n) / O(log n) (problem forbids)

## Pitfalls
- After swap with `hi`, do **not** advance `mid` (the new value at `mid` is unknown)
- After swap with `lo`, **do** advance `mid` (we know the new value at `mid` was `1` from `[lo..mid)`)
- Loop is `mid <= hi`, not `mid < hi`
