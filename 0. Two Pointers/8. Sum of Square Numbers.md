# 8. Sum of Square Numbers (LC 633) — Medium
🔗 [Open on LeetCode](https://leetcode.com/problems/sum-of-square-numbers/)

**Pattern:** Two Pointers — range two-pointer over numeric range `[0, √c]`
**Trigger:** decide existence of `a² + b² == c` (or `a*b`/numeric pair on a bounded range)

## Problem (1-liner)
Given non-negative `c`, decide if `∃ a, b ≥ 0` with `a² + b² == c`.
**Example:** `c=5` → `true (1²+2²)`; `c=3` → `false`

## Idea (1-liner)
`a` walks up from `0`, `b` walks down from `√c`. Compare `a²+b²` to `c` and step accordingly.

## Approach
1. `a = 0`, `b = ⌊√c⌋`.
2. While `a <= b`:
   - `s = a*a + b*b`
   - `s == c` → `true`
   - `s < c` → `a++`
   - else → `b--`
3. Return `false`.

## Code (clean canonical)
```cpp
bool judgeSquareSum(int c) {
    long long a = 0, b = (long long)sqrt((double)c);
    while (a <= b) {
        long long s = a * a + b * b;
        if (s == c) return true;
        if (s < c) a++;
        else b--;
    }
    return false;
}
```
**TC:** O(√c)  **SC:** O(1)

## My Solution
```cpp
class Solution {
public:
    bool judgeSquareSum(int c) {
      if (c <= 1) return true;
      long long start = 0, end = (long long)sqrt((double)c);
      while(start <= end) {
        long long sumOfSqure = (start * start) + (end * end);
        if(sumOfSqure == c) return true;
        else if(sumOfSqure > c) end--;
        else start++;
      }
      return false;
    }
};
```

## Other Approaches
- **Brute force √c loop:** for `a = 0..√c`, check if `c - a²` is a perfect square → O(√c · log) / O(1)
- **Hash set of squares:** precompute all squares ≤ c, check complement → O(√c) / O(√c)
- **Fermat's theorem on sums of two squares:** `c` expressible iff every prime factor `≡ 3 (mod 4)` appears an even number of times → O(√c) trial division / O(1) (number-theory flex)

## Pitfalls
- `a` must start at `0`, not `1` (handles `c = 0, 1, 4, 9, …`)
- Upper bound is `⌊√c⌋`, **not** `c/2`
- Cast to `long long` **before** multiplication; `c` can reach `2³¹ − 1`
- `sqrt` may round down by FP error — usually safe; defensive: check `b+1` if paranoid
