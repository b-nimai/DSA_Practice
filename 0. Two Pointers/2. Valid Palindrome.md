# 2. Valid Palindrome (LC 125) — Easy
🔗 [Open on LeetCode](https://leetcode.com/problems/valid-palindrome/)

**Pattern:** Two Pointers — opposite ends, skip filter
**Trigger:** check string symmetry / palindrome with character filter

## Problem (1-liner)
Lowercase + strip non-alphanumerics; return `true` if the result is a palindrome.
**Example:** `"A man, a plan, a canal: Panama"` → `true`

## Idea (1-liner)
`l, r` at ends. Skip non-alnum on each side; compare lowercased chars; converge.

## Approach
1. `l = 0`, `r = n - 1`
2. Skip non-alnum: while `l < r && !isalnum(s[l])` → `l++`; same for `r`.
3. If `tolower(s[l]) != tolower(s[r])` → `false`.
4. Else `l++, r--`.
5. Loop ends → `true`.

## Code (clean canonical)
```cpp
bool isPalindrome(string s) {
    int l = 0, r = s.size() - 1;
    while (l < r) {
        while (l < r && !isalnum(s[l])) l++;
        while (l < r && !isalnum(s[r])) r--;
        if (tolower(s[l]) != tolower(s[r])) return false;
        l++; r--;
    }
    return true;
}
```
**TC:** O(n)  **SC:** O(1)

## My Solution
```cpp
class Solution {
public:
    bool isPalindrome(string s) {
        int start = 0, end = s.size() - 1;
        while(start < end) {
          char ch1 = s[start];
          if (!isalnum(ch1)) { start++; continue; }
          char ch2 = s[end];
          if (!isalnum(ch2)) { end--; continue; }
          if(tolower(ch1) != tolower(ch2)) return false;
          start++, end--;
        }
        return true;
    }
};
```

## Other Approaches
- **Build cleaned string + reverse compare:** filter to new string, compare with `reverse` → O(n) / O(n)
- **Build cleaned string + 2-ptr:** same prep, then 2-ptr → O(n) / O(n)
- **Recursion:** check ends, recurse inward → O(n) / O(n) stack

## Pitfalls
- Inner skip loops must also check `l < r` (else you walk past)
- Use `isalnum` and `tolower` from `<cctype>` (cast to `unsigned char` if locale-strict)
- Empty string / all-symbol string is a palindrome → `true`
